## Generating Variants
For each variant:
1. Create the directory: mkdir -p .aesthetic/variants/cycle-{N}/
2. Write the variant file: .aesthetic/variants/cycle-{N}/variant-{a|b|c|d}.tsx
3. Each variant is a complete, self-contained component file
4. Variants must be meaningfully different from each other — different layout, different spatial approach, different motion patterns

## TypeScript Verification (MANDATORY)
After writing each variant:
1. Create temp directory: mkdir -p /tmp/et-aesthetic-verify/src
2. Copy the variant to the target path: cp .aesthetic/variants/cycle-{N}/variant-X.tsx /tmp/et-aesthetic-verify/[target_path]
3. Copy necessary files for compilation context
4. Run: cd /Users/manny/dev/emanuelteklu && npx tsc --noEmit
   (temporarily swap the file, verify, then restore)
5. Record result in .aesthetic/variants/cycle-{N}/variant-X.verify.json:
   {"passed": true, "errors": []}
6. If errors: fix the variant and re-verify. Do NOT mark ready with failing variants.
